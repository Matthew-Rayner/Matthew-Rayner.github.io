Plague Frontier

Plague Frontier is a wave-based zombie shooter developed in Unity, featuring a weapon system with attachments, wave based scaling, and fun physics interactions.

The project was built as part of my MSc in Game Development using Unity and C#. I was in a group with one other student so I concentrated on programming while he used asset packs and designed the level, UI and story elements. I focused on enemy AI, game balancing, and satisying gameplay. The game includes a wave spawning system, wave based enemy scaling, and an expandable weapons system.

Some of the systems I developed include:

  • Enemy wave management system that tracks all enemies and makes them stronger each round
  • A custom editor tool for creating spawnpoints
  • An highly configurable and expandable weapon system
  • A project wide audio system
  • A pose swapping system for aiming down sights using Unity's Animation Rigging package

My main takeaways from this project were learning how to:

  • Create an audio system with an external settings file
  • Work with a rigged character and animations
  • Implement navmesh based navigation in an environment with verticality
  • Work with an asset pack effectively
  • Implement a weapon system with sfx and vfx for player feedback
  • Implement an expandable event system for events like "Shot" or "Interacted" using UnityEvents and delegates
