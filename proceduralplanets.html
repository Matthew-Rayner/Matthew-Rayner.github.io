Procedural Planet Generation Tool

The Procedural Planet Generation Tool is a tool built to be used by game developers to generate stylized looking planets.

The project was built as part of my MSc in Game Development using Unity and C#. For this module, we were given freedom to make whatever we wanted so I chose to work with procedural terrain. After talking to my supervising lecturer I began the 12 week journey that was this project.

Some of the major challenges in this project were:

  • Optimising the generation algorithm
  • Sewing up seams of the shape
  • UV mapping a procedurally generated object
  • Creating a system to save the objects once generated

My main takeaways from this project were learning how to:

  • Generate terrain procedurally using perlin noise
  • Create meshes through code
  • Generate UVs and Textures for meshes
  • Create a basic water shader with shadergraph
